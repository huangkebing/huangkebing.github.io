三、基础接口

BeanMetadataElement，只有一个方法，实现这个接口就具备了返回配置源的能力：

方法 说明
Object getSource() 返回此元数据元素的配置源 {@code Object}（可能是 {@code null}）

AttributeAccessor，属性访问器，实现该接口就具备了储存、访问元素的能力：

方法 说明
void setAttribute(String name, @Nullable Object value); 添加元素，value可以为null
@Nullable Object getAttribute(String name); 获取元素，可能返回null
@Nullable Object removeAttribute(String name); 移除并返回，可能返回null
boolean hasAttribute(String name); 判断是否存在
String[] attributeNames(); 获取所有元素名

这两个接口是spring-core中的，是两个基础接口，应该是有很多实现类
